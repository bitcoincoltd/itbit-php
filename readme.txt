Title: itbit-php Classes
Version: 1.0.1
Release Date: 2015-04-06
Author: David Barnes (david.barnes (at) bitcoin.co.th

Description:
This PHP class allows you to interact with the itbit.com API.
In order to use the API you must email api@itbit.com (including your PGP public key) to get your client ID and secret.
Once you have this information you can plug it into the class to purform the following operations

* Get wallets
* Get balances
* Create orders
* Cancel orders
* Withdraw funds
* Get deposit address
* Get trade history
* Get orders
